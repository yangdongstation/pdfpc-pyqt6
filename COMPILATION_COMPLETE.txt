üìû TROUBLESHOOTING
================================================================================

If executable won't run:
  1. Check file permissions: chmod +x pdfpc-pyqt6/pdfpc-pyqt6
  2. Check dependencies: ldd pdfpc-pyqt6/pdfpc-pyqt6 | grep "not found"
  3. Check DISPLAY: echo $DISPLAY (for X11/Wayland)
  4. See BUILD.md for detailed troubleshooting

If size is larger than expected:
  1. Verify Pillow is excluded: ls _internal/PIL (should not exist)
  2. Check for unnecessary modules: ls _internal/PyQt6/
  3. Try UPX compression for further reduction

If performance is slow:
  1. This is normal for first PDF load (pages are being rendered)
  2. Second and subsequent loads use cached images
  3. Preload images in background (configured automatically)
